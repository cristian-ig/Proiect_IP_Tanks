# Proiect_IP_Tanks
Proiect IP, Tema: Tanks, Membri: Cristian Ignat Ionut Ciobanu


What you need before download:
-SDL2
-SDL_mixer
-GLEW
-GLM

Anyway, there should be everything you need, including those (in the folder 'Desp')

/If having trouble with linkers, link the library from the 'Desp' folder/

Controls:
Player 1 : W, A, S, D, SPACE, E, Q and Mouse (single player only)
Player 2 : UP, DOWN, LEFT, RIGHT, NUM-1, NUM-2, NUM-3
